Batch Mailer

Overview
Lets you batch email your accounts. This is a basic mailer. If you have a lot of accounts, it is recommended you use a utility better suited to batch mailing.
Subject
Enter subject of email
From Name
Enter from name
From Email
Enter from email. Will also be the reply to email.
Message
Enter message body. Plain text only. HTML version will be created from text. The following tags may be used:

{NAME} = Account Name
{WEBSITE} = Website Name
{WEBSITE_URL} = Base url
Accounts
Specify which account to send message to.
Batch Mailing Options
There are many batch mail options you can utilise on the web. From mailing list software to free bulk utilities, try a search and you`ll find all sorts of options. I can`t recommend anything personally.
