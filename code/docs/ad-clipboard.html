Sale Clipboard

Overview
Lets you copy sales items to a clipboard for retrieval on the add new sale page. The data is stored in the database, so if you log out and back in, the clipboard will retain the data.

Options
To delete an entry click the icon. To clear all, the clear all link.

Click the icon to select download price or physical CD price. Appears ONLY if entry is collection.
Add to Sale Clipboard
You can add to the sale clipboard on the collection and manage tracks pages.
