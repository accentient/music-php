Add New/Edit Tracks

Overview
Enables you to add tracks to collection or edit them depending on which option you clicked.

DO NOT have foreign characters in file names as they won`t be recognised by the system. PHP doesn`t handle unicode filenames across operating systems too well. If you don`t see your file listed, rename it!
Select Folder
On initial load you will be asked to choose a main folder. An example screenshot is shown below:

Track Info
One tab is shown per track with initial file name displayed. Maian Music will attempt to read the tag data, but this may not always be possible if the information doesn`t exist. In this case, enter data manually. Example screenshot below.


Enter information for each track and navigate via the numeric tabs. No information is lost when clicking new tabs. Click the "Add This Track" checkbox to include this track when adding tracks. At least 1 MUST be selected. When you check / uncheck the tabs will change colour so you can easily identify tracks which are checked and which are not.


Use the "Toggle All" feature to quickly check /uncheck all. Note that if you have one track checked, when you toggle, this will be unchecked, but the rest will be checked. The "Toggle All" always reverses the current selections.
Disable Track
To disable a track leave the cost field blank. Entering 0.00 or 0 will assume the track is available and it`s free.
Preview File
To add a preview, start typing a keyword in the preview box and an auto complete option will appear with search results. Click on the track which is your preview. Screenshot example below:



When the preview file has loaded you can click the icon to play the track.

Note that the auto complete only starts after min 4 characters are typed, to change this edit the 'AUTO_COMPLETE_MIN_LENGTH' value in the 'control/defined.php' file.
