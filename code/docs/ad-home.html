Admin Homepage

Overview
Main screen entered on admin login
Latest XX Sales
Defaults to 10 and shows the latest 10 sales. To increase this value, edit the following value in the "control/defined.php" file:

define('ADMIN_HOME_LATEST_SALES', 10);
For a description of the sales functions, see the sales section here.
Latest XX Accounts
Defaults to 10 and shows the latest 10 accounts. To increase this value, edit the following value in the "control/defined.php" file:

define('ADMIN_HOME_LATEST_ACCOUNTS', 10);
For a description of the account functions, see the account section here.
