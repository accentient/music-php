Main Features
HTML5 Responsive System
MOSS compliant

Group music into collections
Add offers and gift coupons
Manage via attractive admin interface
Email license agreement and checkout terms and conditions
Supports Disqus comments system
Supports Facebook OG meta tags

Supports AddThis for social bookmarking

RSS feeds

Basic shipping and tax options
Buyer account system for order overview
Download controls and restrictions for downloads
SEO friendly
Add and manage new pages
Track ordering via drag and drop
