Collections

Overview
Music files are grouped into collections. A collection can be a set album/CD or a random group of tracks.

To add new collection click the button.

To search collections click the button. This opens the search panel.

Add Tracks

Add Collection to Sale Clipboard
To add a collection to the sale clipboard click the icon. If you are adding a sale to the system manually, you can add items (collections and/or tracks) to the sale clipboard. On the add sale page you select which items you wish to add to the sale.
