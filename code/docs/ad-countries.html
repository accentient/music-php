Tax
Taxes are shown as follows:

T = Tangible tax
D = Digital download tax

"Off" denotes the values for the country is set as "no", which switches off tax for that country. If the country tax is blank or 0, then it will revert to the settings default. Setting a value for the country tax overwrites the default.
Flag Display
For a flag to display the image MUST be named with the countries iso code, have a PNG extension and exist in the following directory:

admin/templates/images/flags/
See existing flags for reference.
