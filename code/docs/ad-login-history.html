Login History

Overview
Enables viewing of account login events. Useful to see if accounts are being shared.
Add New Account
To add new account click the button.
Export History
To export history to CSV click the button. Note that if the date filter is used, export will mirror the filter.
Change Date Filter
To filter by date range click the button. This opens the date panel.
Delete
To delete history, click the icon.
Delete ALL
To delete ALL entries click the button.
