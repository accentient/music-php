Add New Style

Enter Style Names. One per row
Enter new style names, one per row. Example:

Style One
Style Two
Style Three
A second, optional pipe delimited parameter may also be set, which is the custom slug if applicable. You MUST specify TWO pipes to delimit. Example:

Style One||slug-style-one
Style Two||slug-style-two
Style Three
Not all need to be specifed and the slugs can be edited later.
Style Type
Specify if style is a top level style or sub style of existing style. Note that a top level style is simply a placeholder that toggles sub styles. You cannot add collections to top level styles if it contains sub styles.
Enable Style(s)
Yes to enable the style, No to disable the style.
Link to Collection
This is a useful feature if you prefer to have a collection load when a style category is clicked. So, instead of showing all collections associated with that style it would load the specified collection.

To locate a collection, start typing in the box to initiate the auto complete. When you see the desired collection, click on it to set it. Use the delete icon to remove it. Note that the association isn`t complete until you add or edit a style.

Note that you cannot link a collection if the style is a top level style. If the top level is selected the "Link to Collection" tab is hidden.

Note that the auto complete only starts after min 4 characters are typed, to change this edit the 'AUTO_COMPLETE_MIN_LENGTH' value in the 'control/defined.php' file.
