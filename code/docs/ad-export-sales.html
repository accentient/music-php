Documentation

Export Sales

Overview
Enables you to export sales to CSV.
Sale Date - From
Enter sale 'from' date. Clicking into box launches calendar.
Sale Date - To
Enter sale 'to' date. Clicking into box launches calendar.
Sale Status
Filter by status. Hold down Ctrl key for multiple selections.
Payment Gateway
Filter by payment gateway. Hold down Ctrl key for multiple selections.
Include Music Purchases
If set to yes, includes music purchase data in CSV export.
Tangible Tax Country / Digital Tax Country
Filter sales by tax countries.
Filter By Account
If you wish you can filter by account. Typing into the box launches the auto complete option. When you see the selection you want, click to load into box. Click the icon to remove. See the following screenshots:





Note that the auto complete only starts after min 4 characters are typed, to change this edit the 'AUTO_COMPLETE_MIN_LENGTH' value in the 'control/defined.php' file.
