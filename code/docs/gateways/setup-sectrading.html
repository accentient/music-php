Secure Trading - Setup Step 1
1: Sign up for a Secure Trading account to get your site reference and merchant details.
2: Log into your Secure Trading account. Click "Notifications > Add Filter" from the left hand menu. Enter description for new filter, check "Auth" in requests and select ALL payment types. Then save new filter. Next click "Add Destination". Complete the options as follows:

Description: Any name
Notification type: url
Process notification: offline
Destination: Callback url as shown below
Notification password: Unique password different to merchant password
Security algorithm: sha256

Check ALL field boxes and in the "custom fields" box enter and add the field "custom-data". Then click to save destination. Finally, on main notifications screen assign the filter you created to the destination you created and click save. Make sure the "active" box is checked.

3: Still in your Secure Trading account area, click "Payment pages redirects" from the left menu. Click to create a new condition. The default checked boxes should be fine, so just name the condition. Next, create an action, and enter the action url as shown below. Click save. On the main page, assign the action to the condition. Make sure the "active" box is checked.
4: Enter applicable details in your Maian Music admin area:

Payment Gateways > Secure Trading

Once you are happy everything is working, contact Secure Trading to switch your account to live.
Secure Trading - Callback Url
The callback url for all operations is:

http://www.example.com/callback/sectrading.php
https://www.example.com/callback/sectrading.php (SSL)

Where 'example.com' is your own domain.
Secure Trading - Action Url
The action url for all operations is:

http://www.example.com/?msg=8
https://www.example.com/?msg=8 (SSL)

Where 'example.com' is your own domain.
