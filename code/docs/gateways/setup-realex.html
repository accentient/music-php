Realex Payments - Setup
1: Sign up for a Realex Payments account to receive your merchant account information and secret key.
2: Email Realex Payments and tell them to set the response/callback url for your account to the 'callback/realex.php' file in your installation. Give them the FULL url:

Example:
http://www.example.com/callback/realex.php (Or https if over SSL)

(Note. This cannot be currently done in your Realex control panel).

3: Enter applicable details in your Maian Music admin area:

Payment Gateways > Realex Payments
Realex Payments - Callback Url
The callback url for all operations is:

http://www.example.com/callback/realex.php
https://www.example.com/callback/realex.php (SSL)

Where 'example.com' is your own domain.
Realex Payments - Sandbox
Realex Payments do have a test system and will be more than happy to set up a test account for you. Contact them to set up the test account. This will involve signing and returning some documentation. Their service is fast and efficient.
