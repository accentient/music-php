IcePay - Setup
1: Sign up for an IcePay account and activate via email registration.
2: Log into your IcePay account. Click "My websites" from the menu.
3: For the thank you page enter: http://www.example.com/music-store/?msg=8 (Or https if over SSL)
4: For the error page enter: http://www.example.com/music-store/?msg=1 (Or https if over SSL)
5: For the postback url enter: http://www.example.com/music-store/callback/icepay.php (Or https if over SSL)
    (Make sure the paths reflect your OWN website)
6: Once complete, return to the "Edit website" tab to view your merchant ID and secret/encryption code.
7: Enter applicable details in your Maian Music admin area:

Payment Gateways > IcePay
IcePay - Callback url
The callback url for all operations is:

http://www.example.com/callback/icepay.php
https://www.example.com/callback/icepay.php (SSL)

Where 'example.com' is your own domain.
IcePay - Sandbox
When you add a website to IcePay (as above), it defaults into test mode. If you are then planning to go live with IcePay you`ll need to follow the instructions in the registration email and upload the requested documents. Until this is done, your account remains in test mode.
