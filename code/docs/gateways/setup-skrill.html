Skrill - Setup
1: Sign up for a Skrill account.
2: Access your Skrill account.
3: Click "Merchant Tools" (If you don`t see this, ask Skrill to enable it).
4: Enter a "Secret Word".
5: Save settings.
6: Enter applicable details in your Maian Music admin area:

Payment Gateways > Skrill
Skrill - Callback Url
The callback url for all operations is:

http://www.example.com/callback/Skrill.php
https://www.example.com/callback/Skrill.php (SSL)

Where 'example.com' is your own domain.
Skrill - Sandbox
Skrill doesn`t have a sandbox immediately available, but you should contact Skrill and get them to set you up two test accounts, a personal account and a merchant account. Then you will easily be able to test the system.

Test mode should be enabled before going live. This is enabled/disabled in your admin area.

Settings > Payment > Payment Mode
