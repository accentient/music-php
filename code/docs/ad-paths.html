Check Paths

Overview
Here you can check your music files and cover images exist in the system in the specified directories.

IMPORTANT! If the system reports ALL files are missing, your paths may be wrong in the settings OR it may be a case issue. For example .JPG and .jpg are NOT the same and would cause issues on Linux servers.
Check Paths
To check paths, simply click the button.

The following will be checked:

MP3 File Path
Preview MP3 File Path
Cover Art Path

Collections with no cover art will not be reported as missing!
Music tracks with no mp3 preview file will not be reported as missing!
Music tracks with no mp3 file WILL be reported as missing!

A limit of 200 entries for each section is default. This is to prevent everything from being reported as missing if, for example, you change the base folder names by mistake. If you wish to increase the limit change the 'PATH_CHECK_LIMIT' value in the 'control/defined.php' file.
