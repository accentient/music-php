Statistics

Overview
Gives you a statistical overview of sales.
Default Settings
To manage the default settings click the button. In the default settings, the following are available:

Default Years for Comparison (Comma Delimit) - Enter 2 years, comma delimited.
Default Display for Best Sellers - Numeric option. You can edit the options in the 'content/language/admin.php' language file.
Default Display for Month Filters - Default of this month or last month.
Legacy Duration for Month Filters - For reload drop down. Defaults to 12 months back.

Monthly Revenue Overview
Displays a revenue overview of monthly sales. Click the "Reload" button to select a different month. Note that sale revenue is based on sub totals and doesn`t include shipping or tax costs etc. So the total of goods before additions/subtractions. Hover your cursor over the data bars to see the values. See above screenshot.
Best Selling Collections
Displays a list of the best selling collections. Number shown is the amount of collections sold, this includes CDs. Click the "Reload" button to select a different amount to display.
Best Selling Tracks
Displays a list of the best selling tracks. Number shown is the amount of tracks sold. Click the "Reload" button to select a different amount to display.
Sales By Country
Displays sales totals by country. A sale is one unit, regardles of items purchased. Note that this function won`t display if the GeoIp functions are disabled in the settings as this feature will not work or be accurate.

A threshold groups countries that are less than 3%, these will be shown as "other". If you wish to increase or remove this threshold edit the "admin/templates/js/mm-stats.js" file and line 50:

threshold: 0.03
Value can be 0 to 1. The "combine" parameter can be removed completely if required. Further reference can be found here.

If you cannot read the countries because of the colour, hover your cursor for a better readable reference. Example:

Sales By Gateway
Displays sales totals by payment gateway. A sale is one unit, regardles of items purchased.

A threshold groups gateways that are less than 3%, these will be shown as "other". If you wish to increase or remove this threshold edit the "admin/templates/js/mm-stats.js" file and line 18:

threshold: 0.03
Value can be 0 to 1. The "combine" parameter can be removed completely if required. Further reference can be found here.

If you cannot read the gateways because of the colour, hover your cursor for a better readable reference. See countries example above.
