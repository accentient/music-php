Add New Offer

Discount
Enter discount. Can be fixed amount or percentage. Examples:

2.99
15%
Applies To
Discount can apply to downloadable options only, cds only or track downloads only. If none are set, defaults to all types.
Restrict to Collections
If this discount applies only to items in certain collections, use the boxes to check. If none are set, discount applies to all collections.
Expiry Date
Expiry date for discount if applicable. Clicking into box launches the calendar.
Enable Offer
Yes to enable, No to disable
