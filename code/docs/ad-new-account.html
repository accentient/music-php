Add New Account

Overview
Lets you manually add a new account to the system. When a sale occurs, accounts are auto created if it doesn`t already exist.
Name
Enter account name or alias. Max 250 chars.
Email
Enter account email address. Must be unique.
Country of Residence (for Tax)
For tax purposes and to comply with MOSS guidelines for E.U countries.
Password
Enter password. Click the icon to auto generate password. Passwords should be as secure as possible. Using a combination of alphanumeric characters and special characters is recommended. DO NOT make it easy for hackers/bots.
Send Account Info Email
Appears only when adding account. If checked, sends account details to email address.
Shipping Method
Default shipping method. For CDs. Optional.
Address 1
Shipping address field. For CDs. Optional.
Address 2
Shipping address field. For CDs. Optional.
City
Shipping address field. For CDs. Optional.
County/Region/State
Shipping address field. For CDs. Optional.
Post/Zip Code
Shipping address field. For CDs. Optional.
Country
Shipping address field. For CDs and tax purposes.
Timezone
Account timezone. Defaults to main timezone in settings if not set.
IP Address
IP address if known. If blank will be updated on first login.
Date Created
Date account was created.
Bypass Download Restrictions
If set to yes, any download restrictions set in the settings will be bypassed.
Enable Login Events
Yes to enable, No to disable
Enable Account
Yes to enable, No to disable
