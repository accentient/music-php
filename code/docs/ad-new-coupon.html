Add New Coupon

Code
Enter preferred coupon code. Max 30 chars. Codes must be unique.
Discount
Enter discount value. This applies to the sub total ONLY and doesn`t include discounts on shipping or tax. Enter monetary value or percentage. Monetary value with no currency symbol. Examples:

5.99 = Discount of 5.99 on sub total. 10.00 sub total would reduce to 4.01
25% = 25 percent discount of sub total. 10.00 sub total would reduce to 7.50.
Expiry Date (Optional)
Enter expiry date if required. Coupon will expire on this date. Clicking into box will show a calendar for date selection.
