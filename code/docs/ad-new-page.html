Set As Landing Page
The landing page is the page people will see on accessing the root url of you music store. Sometimes called the homepage. If you set a new page as the landing page, this overwrites the main system homepage.
Enabled
Yes to enable, No to disable.
Custom Template
Custom templates must be created in the following directory with the .tpl.php file extension:

content/*THEME*/custom-pages/
Where *THEME* is the theme you are using.

See the notes in the 'content/*THEME*/custom-pages/test.tpl.php' file for more information.

A contact form is included in Maian Music, for this to load on any page set the custom template to 'contact.tpl.php'. If you need a more complex form, copy this file and make the necessary changes.

Meta Description

URL Rewrite Slug
For SEO. Your server must support rewrite rules via htaccess. Enable in settings. If no custom slug is set, system will create one from the page name. If your server doesn`t support this you may see a 500 Internal Server Error.
